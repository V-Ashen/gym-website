# Fitness Sports Center - Promotional Website

This is a responsive promotional website built for a local fitness brand, created as part of the Web Development Internship Evaluation for Koncept Hive.

# Live Demo & Design
- Live Website: https://gym-website-wheat-two.vercel.app/
- Figma UI Design: https://www.figma.com/site/OPnN67WpQVYNROcbyDwL8S/Untitled?node-id=0-1&t=3U2Utgs7AgeqqkQC-1

# Tech Stack
- Frontend Framework: React (Vite)
- Styling: Tailwind CSS v4
- Deployment: Vercel
- Version Control: Git & GitHub
